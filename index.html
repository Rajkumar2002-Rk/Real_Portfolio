Insurance Fraud Detection System

AWS Lambda | SageMaker | Kinesis | Python

Developed a cloud-native fraud detection platform using AWS S3 and SageMaker. Implemented Random Forest classifiers to detect anomalies in real-time.

The Challenge

Insurance claims verification was manual and slow, leading to delays and potential fraud slipping through.

The Solution

I architected a serverless pipeline on AWS. Raw claims data is ingested into S3, triggering a Lambda function for pre-processing. The data is then fed into a SageMaker endpoint running a Random Forest model to classify the claim as 'Legitimate' or 'Suspicious' in real-time.

The Result

The system achieved ~90% accuracy in detecting anomalies, significantly reducing false positives and optimizing the claim validation process.

3D Face Generation with NeRF

TensorFlow | JAX/Flax | Computer Vision

Built an AI pipeline to generate photorealistic 3D faces from video clips using Deformable Neural Radiance Fields (NeRFs).

The Challenge

Standard 3D rendering struggles to capture non-rigid facial dynamics (like smiling or blinking) from simple 2D video inputs.

The Solution

I implemented Deformable Neural Radiance Fields (NeRFs) to model these complex dynamics. I built the training framework using JAX/Flax and TensorFlow and integrated tools like COLMAP and OpenCV for data preprocessing.

The Result

The model achieved high-fidelity synthesis validated by PSNR metrics, outperforming baseline NeRF models in both training speed and inference quality.

Customer 360 Data Integration (AEP)

Adobe Experience Platform | XDM | ETL

Engineered an end-to-end pipeline unifying customer interactions from CRM, web, and mobile into a single Real-Time Profile.

The Challenge

The Solution

I designed XDM schemas to standardize data and configured Identity Stitching to link these disparate sources. I built ingestion pipelines using both batch and streaming connectors to feed the Real-Time Customer Profile.

The Result

Enabled the marketing team to activate audiences across Adobe Target and Meta Ads based on a trusted, 360-degree view of the customer.
